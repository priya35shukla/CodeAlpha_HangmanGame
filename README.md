
## CodeAlpha Hangman Game

## Description

This is a simple Hangman Game developed in Python as part of the CodeAlpha Internship Program. The game allows the user to guess a hidden word letter by letter within a limited number of attempts.

## Features

- Random word selection
- User-friendly interface
- Limited attempts for guessing
- Displays guessed letters
- Win and lose conditions

## Technologies Used

- Python 3

## How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:

python hangman.py
